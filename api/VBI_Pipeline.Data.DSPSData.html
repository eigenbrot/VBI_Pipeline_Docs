DSPSData¶

class VBI_Pipeline.Data.DSPSData(data_dir, target, suffix='FITS')[source]¶

Bases: object

DSPSData is designed to handle the aggregation of a bunch of files that belong to multiple DSPS Repeat numbers of VBI. This class reads all files in a directory and populates a dictionary that maps DSPS number to a list of filenames. It can then create and return instances of ObservationData based on this dictionary. In this way the entire directory only needs to be read once, which saves a lot of time.

Initialize a few parameters and read the directory.

Parameters
data_dirstr

Path to directory in which to look for data

targetstr

The VBI target to look for

suffixstr

Optional file mask. The full mask is DATA_DIR/*SUFFIX.
