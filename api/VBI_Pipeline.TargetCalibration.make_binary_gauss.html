Convert an array to a binary image using a gaussian smoothing method.

This method is designed to highlight features with high spatial frequency (i.e., grid lines). The algorithm is:

  1. Create a smooth version of the input data
  2. Subtract the smooth and un-smooth data
  3. Use Otsuâs method to threshold the resulting image
  4. Apply the threshold to the difference image.

The idea here is that the interesting features have been smoothed out of the smooth image and thus the difference image has very strong signal at these features. To work properly the amount of blur should be very close to the pixel width of the features to be extracted.

Parameters:

data : numpy.ndarray

Input data array

sigma : float

Width (in px units) of smoothing kernel. See above; this should be close to the size of the feature to be extracted.

binary_threshold : float

Min percentage of pixels to be on one side of the Otsu threshold when constructing the binary image. If the percentage of pixels above the threshold is outside the range (binary_threshold, 1 - binary_threshold), then some cleaning of outlying pixels is performed.

sigclip : float

Any pixels beyond sigclip standard deviations from the mean are excluded from the thresholding calculation.

debug : bool

If True, write out the binary image for later inspection. It will be called âbinary.fitsâ
Returns:

numpy.ndarray

The binary image
