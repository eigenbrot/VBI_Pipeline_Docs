main¶

VBI_Pipeline.DarkCalibration.main(dark_dir, outname, low_acceptance_threshold=0.318, high_acceptance_threshold=0.318, suffix='.fits')[source]¶

Run a raw data through the complete Dark Calibration pipeline.

The result will be a Dark Calibration object suitable for use in other Calibration pipelines.

Parameters
dark_dirstr

Location of directory containing the raw dark frames to use.

outnamestr

Where to save the resulting Dark Calibration FITS file

[low|high]_acceptance_parameterfloat

If the fraction of pixels in the final object that are less than 1 stddev away from the mean is outside the range (0.682 - low_acceptance parameter, 0.682 + high_acceptance_parameter) an error will be thrown.

suffixstr

Only look for data with this suffix

Returns
Data.FitsData

Completed Dark Calibration
