fwhm¶

VBI_Pipeline.TargetCalibration.fwhm(x, y)[source]¶

Simple function to fit Hough peaks that have flat tops or are otherwise non-gaussian.

Very simple: just return the center of the line defined by the FWHM

Should work well on Gaussian peaks, too.
