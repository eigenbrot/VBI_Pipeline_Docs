average_FitsData¶

VBI_Pipeline.generic.average_FitsData(fitsData, preserve_camera_pos=True)[source]¶

Average a Data.FitsData object

In the most general case this will simply collapse along the last axis of the stacked data (i.e., collapse all hdus). However, there is also the option to preserve VBI camera positions, in which case each camera position is collapsed separately.

In the first case the header information is taken from the first data hdu, and in the second case the header for each camera position is taken from the first data hdu at that position.

Parameters
fitsDataData.FitsData

Should probably actually have some data in it

preserve_camera_posbool

If True then keep the separate camera positions separate during the average

Returns
Data.FitsData

The averaged data
