gen_debug_plots¶

VBI_Pipeline.TargetCalibration.gen_debug_plots(intsects, pos_list, refidx, dmax, c_i, c_ref, min_i, min_ref)[source]¶

Generate 2 debug plots associated with the calculation of the camera shifts:

1. “overlap.pdf” - A plot showing all grid images and all found intersections (yellow). The reference intersections are identified with green dots and all matching candidate intersections have red dots. Colored lines connecting these dots show the intersections identified as being the same. Each candidate image is given a different color.

2. “diffhist.pdf” - A set of histograms for each candidate image. Each histogram shows the distribution of radial distances between ALL intersections in the reference and candidate images. dmax is marked with a vertical line.

NOTE: For this function to work ALL intermediate functions in this module need to have been run with debug=True. This ensures that all needed files exist.

Parameters
intsectslist

Each element of this list corresponds to a single input hdu and contains a numpy array of shape (2, N) containing the (x,y) WCS coordinates for the N intersections found in that hdu. Probably the output of collect_intersections()

pos_listlist

A list containing the camera positions (VBI__005) of each corresponding hdu. Probably the output of collect_intersections()

refidxint

Index of the reference hdu to use. Index corresponds to the location in both intsects and pos_list

dmaxfloat

Any intersections with pixel locations less than dmax apart are considered to be the same intersection.

c_inumpy.ndarray

(2, M) shape array containing (x, y) coordinates of all matching points in the candidate image.

c_refnumpy.ndarray

(2, M) shape array containing (x, y) coordinates of all matching points in the reference image.

min_inumpy.ndarray

1D array containing the radial distance of the closest intersection to all intersections in the candidate image

min_refnumpy.ndarray

1D array containing the radial distance of the closest intersection to all intersections in the reference image
