load_hough_pickle¶

VBI_Pipeline.TargetCalibration.load_hough_pickle(outname)[source]¶

Load a previously-computed set of intersections and position list.

This can save a huge amount of time by bypassing the hough transform steps if all that’s needed is to, e.g., change the value of dmax.

This is a separate function for now (rather than just being in main) because it might have some sanity checks in the future.

Parameters
outnamestr

The path to the final TargetCalibration object. This is used to auto-generate the saved intersection filename.

Returns
lst or None

Each element of this list corresponds to a single input hdu and contains a numpy array of shape (2, N) containing the (x,y) corrdinates for the N intersections found in that hdu. If no file is found then None is returned.

lst or None

A list containing the camera positions (VBI__005) of each corresponding hdu. If no save file was found then None is returned.

str

The name of the loaded/saved intersection/position file. Data are saved with python’s pickle framework.
