get_all_obs_targets¶

VBI_Pipeline.VBIPipeline.get_all_obs_targets(data_dir, suffix='.fits')[source]¶

Find the unique set of data set ID’s (ID___004) for Observation data in a given directory

Only files with DKIST004 == ‘Observation’ will be scanned.

Parameters:

data_dir : str

Path to directory containing data

suffix : str

Optional file mask. The full mask is DATA_DIR/*SUFFIX. Defaults to “.fits”
Returns:

list

A list containing all unique values of ID___004 found in the given directory
