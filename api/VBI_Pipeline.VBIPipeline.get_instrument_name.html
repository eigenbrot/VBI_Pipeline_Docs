get_instrument_name¶

VBI_Pipeline.VBIPipeline.get_instrument_name(data_dir)[source]¶

Given a directory of VBI data, determine which camera was used to take the data.

This function does NOT verify that all of the data are from the same camera (that happens elsewhere), it just returns the instrument (camera) of the first file read by the system.

Parameters
data_dirstr

Directory containing VBI data

Returns:
str

The value of INSTRUME for the first file loaded. Will be ‘VBI-RED’ or ‘VBI-BLUE’
