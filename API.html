VBI_Pipeline.Data Module¶

Classes¶

AlignmentData([data_dir, dataset_ID, suffix])

AlignmentData is an extension of FitData designed to only hold data associated with Alignment. It inherits all of the functionality of FitsData and defines an extra method that loads all data from a directory who's headers match the following: Key Value --- ----- DKIST004 align.

DSPSData(data_dir, target[, suffix])

DSPSData is designed to handle the aggregation of a bunch of files that belong to multiple DSPS Repeat numbers of VBI.

DarkData([data_dir, dataset_ID, suffix])

DarkData is an extension of FitData designed to only hold data associated with Dark. It inherits all of the functionality of FitsData and defines an extra method that loads all data from a directory who's headers match the following: Key Value --- ----- DKIST004 dark.

FitsData([fitsfile])

FitsData is a basically a wrapper for astropy.io.fits, with some extra methods and properties to aid the reduction of DKIST data.

GainData([data_dir, dataset_ID, suffix])

GainData is an extension of FitData designed to only hold data associated with Gain. It inherits all of the functionality of FitsData and defines an extra method that loads all data from a directory who's headers match the following: Key Value --- ----- DKIST004 gain.

ObservationData([data_dir, dataset_ID, suffix])

ObservationData is an extension of FitData designed to only hold data associated with Observation. It inherits all of the functionality of FitsData and defines an extra method that loads all data from a directory who's headers match the following: Key Value --- ----- DKIST004 observe.
